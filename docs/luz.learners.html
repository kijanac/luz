luz.learners module¶

class Learner(trainer, cls, *args, **kwargs)¶

Bases: object

A Learner is an object which takes a dataset as input and produces a predictor as output. Learners are distinguished by the different protocols they use to construct a predictor from a given dataset.

learn(dataset, device, val_dataset=None, test_dataset=None)¶

Learn a predictor based on a given dataset.

Parameters

dataset (Dataset) – Dataset which will be used to learn a predictor.

Returns

Predictor which was learned using dataset.

Return type

luz.Predictor
