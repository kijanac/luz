luz.trainers module¶

class Trainer(loss=None, optimizer=None, start_epoch=1, stop_epoch=2, early_stopping=False, handlers=None, patience=5, **loader_kwargs)¶

Bases: object

Algorithm to train a model using data.

Parameters

  • loss (Optional[Callable[[Tensor, Tensor], Tensor]]) – Loss function to be minimized during training, by default None

  • optimizer (Optional[Optimizer]) – Training optimizer, by default None

  • start_epoch (Optional[int]) – First training epoch, by default 1

  • stop_epoch (Optional[int]) – Last training epoch, by default 2

  • early_stopping (Optional[bool]) – If True, then use val_dataset for early stopping; by default False. Ignored if val_dataset is None.

  • patience (Optional[int]) – Number of epochs of non-improving validation loss before training stops early; by default 5. Ignored if early_stopping is False.

  • handlers (Optional[Iterable[Handler]]) – Handlers to run, by default None.

  • **loader_kwargs – Dataset.loader kwargs.
